### 使用

- 请做为模块在项目中导入使用！
```shell
  # 在你仓库中执行，将本仓库作为模块使用
  git submodule add https://github.com/HChenX/HiMiuiX.git
  # 后续拉取本模块仓库
  git submodule update --init
```
